算法文档编辑器 v1.0.2 Demo

本 Demo 展示 019f4806 会话中的核心功能： 项目管理后台的算法文档编辑与 IP 组操作界面

📁 项目管理

添加/删除项目，选择本地目录，自动创建 code/sf 子目录，数据持久化到 IndexedDB

项目列表 目录选择

📄 算法文档上传

仅支持 .xlsx 格式，自动解析 A-F 列并进行 hex 值格式校验，校验通过才能上传

xlsx校验 确认上传

✏️ IP 组编辑器

每一行参数为一个IP组，支持新增/编辑/删除，全部字段可编辑。保存时校验并写入 xlsx

IP组操作 编辑保存

📊 Code 版本管理

上传代码需关联算法文档和父版本，版本界面展示版本树（含 base 版本文档关联）

版本树 关联追踪
